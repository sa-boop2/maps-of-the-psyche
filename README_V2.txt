# PSYCHE Map Engine - Version 2.0 (Standalone)

This is the updated Version 2.0 of the Psyche Map Engine, reflecting all Phase 8 expansions and refinements.

## What's New in V2:
- **100% Offline Capability**: All external scripts (Three.js) are now bundled locally in the `js/` folder. No internet connection is required for the 3D map to function.
- **Shadow Scanner Expanded**: 8 deep diagnostic questions mapping directly to 16+ frameworks.
- **Deep Dive Grid UI**: Modernized data cards for Neurology, Archetypes, Trauma, etc.
- **Timeline Slider Normalization**: Proportional era-based filtering.
- **Enhanced Atmosphere Engine**: Atmospheric "Internal Weather" (Flow, Anxiety, Depression, Mystical) now respects Light/Dark themes.
- **Procedural Geometries**: Unique 3D shapes for different traditions (Western, Eastern, Indigenous, Esoteric).
- **Consolidated Clean Navigation**: Reverted to fixed top bars for maximum clarity (per request).

## How to Run Offline:
1. **Option A (Simple)**: Double-click `index.html`. 
   *(Note: Some browsers block external data loading on `file://` URLs. If the map doesn't load, use Option B).*
2. **Option B (Recommended)**: 
   - Open a terminal/command prompt in this folder.
   - Run: `python -m http.server 8080`
   - Open your browser to: `http://localhost:8080`
