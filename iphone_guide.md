# Psyche Map: iPhone Installation & Offline Guide

Transform your Psyche Map into a standalone, premium application on your iPhone for deep contemplation anywhere.

## 📱 How to install as an App (iPhone/iOS)

The Psyche Map is designed as a **Progressive Web App (PWA)**. Follow these steps to add it to your Home Screen:

1. **Open Safari**: Navigate to the URL where your Psyche Map is hosted (or open the local `index.html` if you have a local server setup).
2. **Tap Share**: Tap the **Share** button (the square with an upward arrow) at the bottom of the screen.
3. **Add to Home Screen**: Scroll down and tap **"Add to Home Screen"**.
4. **Name it**: Ensure the name is "Psyche Map" and tap **Add**.
5. **Launch**: Find the new "Psyche Map" icon on your Home Screen. It will now open in full-screen mode, without the browser address bar, feeling exactly like a native app.

> [!TIP]
> Using the "Add to Home Screen" version provides a much more immersive experience, allowing the 3D map and custom audio to take center stage.

## 📦 Offline Usage (Standalone ZIP)

Your **Master Edition Masterpiece** has been packaged into a single `psyche_master_offline.zip`.

### How to use the Offline Version:
1. **Extract**: Unzip the file to a folder on your computer.
2. **Run**: Double-click `index.html`.
3. **No Internet Required**: All frameworks, 3D logic, and high-fidelity descriptions are contained within the package.

### Syncing to iPhone (Advanced Offline):
If you want to use it **offline** on your iPhone without a server:
1. Download the **"Documents by Readdle"** or **"Koder"** app from the App Store.
2. Transfer the extracted `psyche/` folder to the app via iCloud or Airdrop.
3. Open `index.html` within that app to run the map offline.

For the full master-edition immersion, always use **Headphones**. The deep-frequency ambient tracks and UI soundscapes are optimized for high-fidelity audio hardware.
