# PSYCHE Map Engine - Version 4.0 (Premium Audio Edition)

This is the feature-complete Version 4.0, featuring a ground-up reconstruction of the audio engine for a high-end, satisfying user experience.

## What's New in V4 (The Premium Audio Overhaul):
- **Organic UI Clicks**: Navigation sounds now feel "thick" and tactile. Instead of flat beeps, we use a three-layer synthesis approach:
  - **Tactile Transient**: A micro-burst of filtered noise for "impact."
  - **Resonant Body**: A weighted sine-wave body with short pitch-drop.
  - **Glistening Finish**: A high-frequency FM chime transient.
- **Evolving Ambient Pads**: The background drone now uses **Super-Oscillators** (detuned pairs) and a **Master LFO-driven breathing filter**. The audio is no longer static; it subtly "washes" and "breathes" over time.
- **Improved Atmospheric Texture**:
  - **Depression**: Deep, dark oceanic rumble with low-pass muffling.
  - **Mystical**: High-frequency crystalline shimmers and celestial pads.
  - **Anxiety**: Tense, dissonant triangle-wave jitter.
- **Enhanced Resonance Visuals**: (From V3) Animated golden lines "shooting" across the void to crystalline octahedrons.

## How to Run Offline:
1. **Option A (Simple)**: Double-click `index.html`. 
   *(Note: Browsers might block external data loading on `file://` URLs. If the map doesn't load, use Option B).*
2. **Option B (Recommended)**: 
   - Open a terminal/command prompt in this folder.
   - Run: `python -m http.server 8080`
   - Open your browser to: `http://localhost:8080`
